🚀 AI-Powered Lead Intelligence Hub
An enterprise-grade automation built in n8n that transforms raw form submissions into actionable sales intelligence.

🧠 The Problem
Sales teams often waste 60% of their time on unqualified leads, leading to "Lead Fatigue" and missed high-value opportunities.

🛠️ The Solution
This workflow automates the entire "Top of Funnel" process:

Data Sanitization: Cleans raw webhook data using custom JavaScript.

Cognitive Scoring: Uses GPT-4o-Mini to analyze budget and business pain.

Structured Persistence: Logs every lead and its analysis into Google Sheets for long-term ROI tracking.

Smart Routing: High-priority leads (Score >80) trigger immediate Slack alerts, while lower-priority leads receive automated Gmail nurtures.

📊 Technical Stack
Trigger: Tally.so Webhooks

Logic: JavaScript (Node.js)

AI: OpenAI API (GPT-4o)

Storage: Google Sheets API

Notifications: Slack API / Gmail API

📥 How to Use
Download the AI Lead Intelligence Hub.json file.

Import it into your n8n instance.

Configure your OpenAI and Google credentials.
